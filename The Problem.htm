THE PROBLEM

 

The problems are identified in the user study as well as my own experience using the Graduate Student Status page. The main problems are summarized from three aspects based on the user tasks (see next section).

About main page

     

Areas to be improved

  • The color is distracting/not emphasizing on the important items/confusions for green red color blindness.
  • The page is very crowded that people don't know where to focus on.
  • The presentation of multiple majors is so easy to be ignored.
  • Display of information is not very organized.
  • Not sure whether graduation has already been applied or not.

About graduation application page

     

Areas to be improved

  • Not showing whether graduation has already been applied or not.
  • Not showing the application summary/users need to go to application to check.
  • Not giving feedback after actions taken such as withdraw, submit or modify application etc.

About POS form

     

Areas to be improved

  • Layout of the page is not very organized.
  • Not giving feedback after actions taken such as save, validate or submit form etc.
  • Not clear showing whether the form has been approved or not.
