Prototyping tool

The prototyping tool I used is InVision in this project. It is a very easy to use prototying tool. My experience using InVision for this project enrich my selections of prototyping tools based on different scopes, time and resources I have.

User scenario

The persona used in the design is for Abby Brown, who is a M.S. student pursuing concurrent majors of Industrial Engineering and Human Computer Interaction. She needs to withdraw her application for Spring 2016 and switch to Summer 2016. Therefore, she wants to check her POS form to see if itâ€™s already completed for graduation.

Supported features/functions

    1. The main page portion of the prototype provides the following functionality:
  • Go to each single major.
  • View information under each major.
  • Hover to see details under requirement session.
  • Go to graduation application page.
  • Go to POS form page.
    2. The graduation application portion of the prototype provides the following functionality:
  • Check application summary on the main page.
  • Go to application form.
  • Withdraw application either in the form or in the summary.
  • Fill out application form/modify application form.
  • Submit application.
    3. The POS form portion of the prototype provides the following functionality:
  • Check information in the POS form.
  • Add courses in the POS form.
  • Save, validate and submit POS form.
